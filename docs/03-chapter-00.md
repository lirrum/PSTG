# 第0章:初识储电

想必各位读者对“储电”这个词并不陌生，目前它已经在整个MC技术圈流行开来，几乎每个MC生电人都知道有这么个东西。本章作为系统学习储电的第一章，并不涉及过多的实践内容，将从储电的定义、历史、圈子、学习准备等等展开叙述。读者需重点关注0.1（储电的定义）和0.3（储电的前置学习）两个小节，其余部分可以先略读，在阅读了前面8章后再回过头来细读。
