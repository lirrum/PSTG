# PSTG简介

PSTG(Plain Storage Tech Guide)是由[Lirrum](https://space.bilibili.com/3546601557592878)编写的MC储电教材。

本教材旨在构建一个系统而全面的储电知识体系，既适合新人系统学习储电，又方便储电研究者查缺补漏。覆盖从基础原理到高阶设计的全维度内容。无论你是刚接触储电的新手，还是希望巩固基础和寻求进阶知识的资深玩家，都能在这本交互式教材中有所收获。
